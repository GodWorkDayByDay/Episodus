EIdFileNotFound
Hierarchy, See Also, Unit: IdTCPConnection
Exception raised when a requested file does not exist.
EIdFileNotFound = class(EIdTCPConnectionError)
Unit
IdTCPConnection
Description
EIdFileNotFound is an EIdTCPConnectionError descendant that represents an exception type raised when the file requested in TIdTCPConnection.WriteFile cannot be found on the local file system.

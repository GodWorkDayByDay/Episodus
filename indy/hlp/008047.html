TQueryType
Unit: IdDNSResolver
Set that contains TQueryRecordTypes values requested for the DNS query.
TQueryType = set of TQueryRecordTypes;
Unit
IdDNSResolver
Description
TQueryType is a set type that may contain one or more values from TQueryRecordTypes. TQueryType is the type used to represent the TIdDNSResolver.QueryRecords property that identifies resopurce records to be examined in the DNS query.

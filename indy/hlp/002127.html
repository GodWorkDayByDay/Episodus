IdBeatsInDay
Unit: IdDateTimeStamp
Beats per day in Swatch Internet Time.
IdBeatsInDay = 1000;
Unit
IdDateTimeStamp
Description
IdBeatsInDay is a constant value that represents the number of Beats in a 24 hour period. Beat time is Swatch's "Internet Time" as described in the reference documents at http://www.swatch.com

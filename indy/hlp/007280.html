TIdDayTimeUDPServer
Hierarchy, Properties, Methods, Events, See Also, Unit: IdDayTimeUDPServer
Implements a DayTime protocol server.
TIdDayTimeUDPServer = class(TIdUDPServer)
Unit
IdDayTimeUDPServer
Description
TIdDayTimeUDPServer implements a UDP-based server for the DayTime protocol (RFC 867). This DayTime protocol is used to retrieve the current day and time in a human-readable format, and is sometimes used for debugging. Use TIdDayTimeUDP for the client connection.

If you need a time-synchronization solution, we recommend that you use the Time protocol encapsulated by TIdTime, TIdTimeUDP, TIdTimeServer, and TIdTimeUDPServer.

TIdDayTimeUDPServer.DefaultPort
TIdDayTimeUDPServer, See Also
Default port number for the server.
property DefaultPort: integer;
Description
DefaultPort is a published property in TIdDayTimeServer that identifies the default port number used by the server, as described in the DayTime protocol specification. The default value for DefaultPort is IdPORT_DAYTIME as assigned in Create.
TIdDayTimeUDPServer.TimeZone
TIdDayTimeUDPServer, See Also
Identifies the Time Zone for the server.
property TimeZone: String;
Description
TimeZone is a String property that indicates the Time Zone where the server is located, and the value is appended to the string representation of the time from the server. The default value for TimeZone is "EST" (Eastern Standard Time), as assigned in Create.

Description
Create is the constructor for the object instance, and creates a new TIdDayTimeUDPServer component. Create sets DefaultPort to the standard port number used for DayTime protocol (IdPORT_DAYTIME), and sets the TimeZone property to "EST" (Eastern Standard Time).

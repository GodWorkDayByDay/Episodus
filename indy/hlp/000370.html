EIdDecompressionError
Hierarchy, See Also, Unit: IdCompressionIntercept
Exception raised when an error occurs while decompressing data using ZLib.
EIdDecompressionError = class(EIdCompressionException)
Unit
IdCompressionIntercept
Description
EIdDecompressionError is an EIdCompressionException descendant that represents the exception type raised when a stream, data, or memory error occurs while decompressing data using ZLib. EIdDecompressionError can be raised in TIdCompressionIntercept.Receive when CompressionLevel is not 0.

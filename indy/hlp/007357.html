Hierarchy, See Also, Unit: IdHash
Abstract base class for hashing algorithm ancestor classes.
TIdHash = class(TObject)
Unit
IdHash
Description
TIdHash is a TObject descendant and the abstract ancestor for classes that implement a specific hashing algorithm to calculate a hash value pf a specific size. TIdHash, at the current time, is an abstract ancestor class that does not have any properties, methods, or events.

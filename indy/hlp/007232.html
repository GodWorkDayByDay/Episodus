TIdCoderCRC16
See Also
Implements a CRC calculation encoder for Tunnel communication classes.
Description
TIdCoderCRC16 is a TIdCoder descendant that provides CRC calculation for encoded message values. TIdCoderCRC16 is used by Tunnel classes like TReceiver and TSender to construct CRC values for encapsulated tunnel message headers.

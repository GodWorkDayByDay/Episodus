Id_TIdFTP_TransferType
See Also, Unit: IdFTP
Default value for the TIdFTP.TransferType property.
Id_TIdFTP_TransferType = ftBinary;
Unit
IdFTP
Description
Id_TIdFTP_TransferType is the default value for the TIdFTP.TransferType property. Id_TIdFTP_TransferType is used in both the class definition and the constructor. This is done to prevent problems with Delphi's form streaming mechanism. Changing this value will change the default value for that property.

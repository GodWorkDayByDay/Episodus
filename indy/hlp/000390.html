EIdIdentHiddenUser
Hierarchy, See Also, Unit: IdIdent
Exception raised when a HIDDEN-USER message is found in an Ident protocol reply.
EIdIdentHiddenUser = class(EIdIdentReply)
Unit
IdIdent
Description
EIdIdentHiddenUser is an EIdIdentReply descendant that represents the exception raised an HIDDEN-USER error is received in the protocol reply in TIdIdent.Reply. EIdIdentHiddenUser will be raised when the protocol reply is parsed in TIdIdent.Query.

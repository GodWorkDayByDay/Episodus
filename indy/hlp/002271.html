Unit: IdIRC.pas

IdIRC.pas contains constants, types, and classes need to implement an IRC (Internet Relay Chat) protocol client, as described in the Internet Standards document:

  • RFC 1459 - Internet Relay Chat Protocol (http://www.rfc-editor.org/rfc/rfc1459.txt)

IdPORT_DAYTIME
Unit: IdAssignedNumbers
Port number for the DayTime protocol.
IdPORT_DAYTIME = 13;
Unit
IdAssignedNumbers
Description
IdPORT_DAYTIME is the constant Integer value used to represent the port number used by the DayTime protocol.

IdPORT_DAYTIME is used by the TIdDayTime and TIdDayTimeServer protocol implementations.

EIdCouldNotBindSocket
Hierarchy, See Also, Unit: IdSocketHandle
Exception when binding a socket descriptor.
EIdCouldNotBindSocket = class(EIdSocketHandleError)
Unit
IdSocketHandle
Description
EIdCouldNotBindSocket is an EIdSocketHandleError descendant raised when TIdSocketHandle.Bind encounters an error while binding the protocol family, address, and port number for the socket descriptor.

EIdPackageSizeTooBig
Hierarchy, See Also, Unit: IdSocketHandle
Exception raised for an invalid datagram size.
EIdPackageSizeTooBig = class(EIdSocketHandleError)
Unit
IdSocketHandle
Description
EIdPackageSizeTooBig is an EIdSocketHandleError exception descendant raised when TIdSocketHandle.SendTo receives the Id_WSAEMSGSIZE protocol stack error after transmission of a datagram.

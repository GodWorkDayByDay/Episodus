IdContainers.pas
Classes
Unit: IdContainers.pas

IdContainers.pas contains classes and methods that provide compatibility with Container classes available in the Enterprise versions of Delphi 4, Delphi 5, and Delphi 6. IdContainers.pas is not required if you use the Enterprise version of these products.

EIdMCastNotValidAddress
Hierarchy, See Also, Unit: IdIPMCastBase
Exception raised when an error occurs during binding allocation.
EIdMCastNotValidAddress = class(EIdMCastException)
Unit
IdIPMCastBase
Description
EIdMCastNotValidAddress is an EIdMCastException descendant that represents an exception type raised when an error occurs while setting or writing to the Multicast group address for a TIdIPMCastBase component. TIdIPMCastBase.IsValidMulticastGroup is used to determine if the group address is a valid multicast address.

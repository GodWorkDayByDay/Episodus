IP_WATCH_HIST_MAX
See Also, Unit: IdIPWatch
Identifies the maximum entries in IP history.
IP_WATCH_HIST_MAX = 25;
Unit
IdIPWatch
Description
IP_WATCH_HIST_MAX is an Integer constant used to define the default value for the maximum number of entries allowed in the hostory list maintained in TIdIPWatch. The default value for IP_WATCH_HIST_MAX is 25.

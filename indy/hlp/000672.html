GFRequestedBlockSize
See Also, Unit: IdTrivialFTP
Default TrivialFTP request block size.
GFRequestedBlockSize = 1500;
Unit
IdTrivialFTP
Description
GFRequestedBlockSize is a constant Integer value that represents the global default block size used in TFTP requests. TIdTrivialFTP.RequestedBlockSize receives the value of GFRequestedBlockSize in the constructor for the object instance.

NoToFacility
See Also, Unit: IdSysLogMessage
Converts an ordinal facility number to its TIdSyslogFacility enumeration value.
function NoToFacility(AFac: Word): TIdSyslogFacility;
Unit
IdSysLogMessage
Parameters
AFac: Word
Numeric facility value to convert to a TIdSyslogFacility value.
Return Value
TIdSyslogFacility - Value for the ordinal facility number.
Description
NoToFacility is a TIdSyslogFacility function used to convert the ordinal facility number in AFac to its TIdSyslogFacility enumeration value.

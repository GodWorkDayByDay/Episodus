TIdThreadClass
See Also, Unit: IdThread
Class type for new thread instances.
TIdThreadClass = class of TIdThread;
Unit
IdThread
Description
TIdThreadClass is a TIdThread class type used by TIdThreadMgr and descendants to represent the class type instance created by the thread manager using CreateNewThread.

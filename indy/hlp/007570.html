TIdMessageFlagsSet
See Also, Unit: IdMessage
Represents message flags currently set in a message instance.
TIdMessageFlagsSet = set of TIdMessageFlags;
Unit
IdMessage
Description
TIdMessageFlagsSet is a Set type that can contain TIdMessageFlags values representing message flags currently set in a message instance. TIdMessageFlagsSet is the type used by the TIdMessage.Flags property.

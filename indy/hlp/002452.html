IdPORT_FTP
Unit: IdAssignedNumbers
Port number for the FTP protocol.
IdPORT_FTP = 21;
Unit
IdAssignedNumbers
Description
IdPORT_FTP is the constant Integer value that represents the default port number used by the File Transfer Protocol.

IdPORT_FTP is used by the TIdFtp protocol implementation.

Unit: IdDummyUnit.pas

IdDummyUnit.pas is a unit that contains only an interface uses clause that includes IdAntiFreeze.pas. It's purpose is to trick the DCC32 compiler into generating .HPP and .OBJ files for run-time units that will not be in the run-time package but will be on the palette in the IDE.

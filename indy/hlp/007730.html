TIdSSLIOHandlerSocket
Hierarchy, Properties, Methods, Events, See Also, Unit: IdSSLOpenSSL
IOHandler for socket-based SSL connections.
TIdSSLIOHandlerSocket = class(TIdIOHandlerSocket)
Unit
IdSSLOpenSSL
Description
TIdSSLIOHandlerSocket is an TIdIOHandlerSocket descendant that represents the socket-based IOHandler used for information, password, and event notifications for a SEcure Socket Layer (SSL) connection. TIdSSLIOHandlerSocket implements properties and methods that allow access to the socket for the SSL transport, and the options used for the connection.

EIdCompressorInitFailure
Hierarchy, See Also, Unit: IdCompressionIntercept
Exception raised when a compression record cannot be allocated for a ZLib stream in TIdCompressionIntercept.
EIdCompressorInitFailure = class(EIdCompressionException)
Unit
IdCompressionIntercept
Description
EIdCompressorInitFailure is an EIdCompressionException descendant that represents the exception type raised if the internal compression record for a ZLib stream cannot be allocated in a TIdCompressionIntercept object instance. The EIdCompressorInitFailure exception will be created with the resource string RSZLCompressorInitializeFailure as the exception message.

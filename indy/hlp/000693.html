GSystemLocale
See Also, Unit: IdGlobal
Character set for the system locale.
GSystemLocale: TIdCharSet  = csIso88591;
Unit
IdGlobal
Description
GSystemLocale is a constant variable that represesents the character set for the system locale. The default value for GSystemLocale is csIso88591 on the Linux platform.

Note: GSystemLocale is defined for the Linux platform only.

IsSymbolicLink
Indicates if a directory entry points to a symbolic link node.
Description
IsSymbolicLink is a Boolean property that indicates if the directory entry is a Symbolic Link code. Use LinkedItemName to access the file or directory name that is the endpoint for the symbolic link. IsSymbolicLink is updated when Parse is used to examine the Data using the TIdFTPListFormat for the directory listing.

EIdFTPServerNoOnListDirectory
Hierarchy, See Also, Unit: IdFTPServer
Exception raised when an event handler has not been assigned for generating directory listings.
EIdFTPServerNoOnListDirectory = class(EIdFTPServerException)
Unit
IdFTPServer
Description
EIdFTPServerNoOnListDirectory is a EIdFTPServerException descendant that represents the exception raised when an FTP server implementation has not assigned an event handler for generating directory listings.

EIdFTPServerNoOnListDirectory is raised in TIdFTPServer when the OnListDirectory event handler has not been assigned for the server implementation.

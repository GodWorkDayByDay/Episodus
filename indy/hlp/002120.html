IdAssignedNumbers.pas
Constants
Unit: IdAssignedNumbers.pas

This unit contains constant values that have been assigned as protocol parameters for fields in the Internet Protocol suite. The Internet Assigned Numbers Authority (IANA) (www.iana.org) has allocated these parameter values for fields in protocols which have been created or are maintained by the Internet Engineering Task Force (IETF). Comprehensive information on assigned numbers can be found in the Internet Standards documents:

  • RFC 1700 - Assigned Numbers (http://www.rfc-editor.org/rfc/rfc1700.txt)

  • IANA Protocol Numbers and Assignment Services (http://www.iana.org/numbers.htm)

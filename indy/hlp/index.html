Welcome to the Internet Direct (Indy) documentation.

Internet Direct (Indy) is an open source internet component suite offering popular internet protocols. Indy is a development library based entirely on blocking sockets, and is available for the Borland Delphi, C++ Builder, and Kylix development platforms.

IdDEF_UseEhlo
See Also, Unit: IdSMTP
Default value for the SMTP client UseEhlo property.
IdDEF_UseEhlo = TRUE;
Unit
IdSMTP
Description
IdDEF_UseEhlo is a Boolean constant that represents the default value for the TIdSMTP.UseEhlo property assigned hte constructor for the object instance. IdDEF_UseEhlo identifies if the SMTP client wishes to use the Enhanced Hello greeting to retrieve supported authentication schemes from an SMTP server.

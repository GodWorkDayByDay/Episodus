FIdPorts
See Also
List containing the defined port numbers for services.
Description
FIdPorts is a TList that is initialized to contain the defined port numbers for well known services on the local computer system, as normally found in the /etc/services file. FIdPorts is used when accessing defined ports using the IdPorts function, as is freeing during finalization for the IdGlobal.pas unit.

EIdTFTPFileNotFound
Hierarchy, See Also, Unit: IdException
EIdTFTPFileNotFound = class(EIdTFTPException)
Unit
IdException
Description
EIdTFTPFileNotFound is a TIdTFTPException descendant that represents the exception haised when the Trivial FTP protocol cannot locate a specified file. EIdTFTPFileNotFound can be raised from TIdTrivialFTP.Get or TIdTrivialFTP.Put.

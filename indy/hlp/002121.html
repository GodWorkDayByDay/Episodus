Unit: IdAuthentication.pas

IdAuthentication.pas contains types, classes and methods, procedure, and functions needed to define and implement an HTTP Authentication framework, as described in the Internet Standards documents:

  • RFC 2616 - Hypertext Transfer Protocol [HTTP/1.1] (http://www.rfc-editor.org/rfc/rfc2616.txt)

  • RFC 2617 - HTTP Authentication: Basic and Digest Access Authentication (http://www.rfc-editor.org/rfc/rfc2617.txt)

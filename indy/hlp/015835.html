IdIMAP4
Classes in Unit IdIMAP4
TIdIMAP4
Implements an Internet Message Access Protocol version 4.1 client.
TIdIMAP4Commands
Represents the enumerated type for IMAP4 commands.
TIdIMAP4ConnectionState
Identifies the connection state, and subsequent valid commands, for an IMAP4 client.
TIdIMAP4SearchKey
Represents a defined search key for the IMAP4 SEARCH command.
TIdIMAP4SearchRec
Represents search criteria for the IMAP4 SEARCH command.
TIdIMAP4StatusDataItem
Represents the defined attributes that can be used in the IMAP4 STATUS command.
TIdIMAP4StoreDataItem
Represents flags for the IMAP4 STORE command.
TIdRetrieveOnSelect
Represents the message data to be retrieved after selecting a mailbox.

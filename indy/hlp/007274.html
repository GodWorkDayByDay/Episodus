SynchronizeEx
See Also
Executes a method in the main application thread.
Description
SynchronizeEx is an overloaded method in TIdCustomThreadComponent that allows the internal thread to execute methods in the context of the main application thread. SynchronizeEx provides additional capabilities not present in Synchronize, including execution of TNotifyEvent methods or TNotifyThreadComponentEvent methods that access Data.

iDEFAULTREPLYBUFSIZE
See Also, Unit: IdIcmpClient
Specifies the default size for ICMP reply buffers.
iDEFAULTREPLYBUFSIZE = 1024;
Unit
IdIcmpClient
Description
iDEFAULTREPLYBUFSIZE is a constant value that specifies the default number of bytes to use as the size for ICMP reply buffers.

TIdServerInterceptOpenSSL
See Also
Implements Secure Sockets Layer support for Indy Server Intercept components.
Description
The TIdServerInterceptOpenSSL server intercept class implements the Open SSL implementation of Secure Sockets Layer through support .DLL's available at http://www.intelicom.si/. You have to deploy these support .DLL's in order to use this component's functionality in your application. To do this, we recommend that you install the .DLL's in the user's system directory.

Indy does not give you any legal rights to use SSL. Some nations including the U.S. regulate or prohibit the export of strong-encryption such as SSL. In addition, using SSL in some nations may be illegal. It is up to you to determine the legal situation in your nation.

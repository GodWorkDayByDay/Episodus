TIdHTTPOptions
See Also, Unit: IdHTTP
Represents the set of HTTP options used for an HTTP client.
TIdHTTPOptions = set of TIdHTTPOption;
Unit
IdHTTP
Description
HTTPOptions is a Set type that can contain TIdHTTPOption values used for a TIdHTTP client. HTTPOptions is the type used to represent the TIdHTTP.HTTPOptions property.

Unit: IdRawHeaders.pas

IdRawHeaders.pas contains Constants, Records, and Types needed to define and implement packet headers and data messages using the various protocol transmissions, including:

  • ARP

  • DNS

  • Ethernet

  • ICMP

  • ICMP ECHO

  • ICMP MASK

  • ICMP UNREACH

  • ICMP TIME EXCEED

  • ICMP REDIRECT

  • ICMP TIMESTAMP

  • IGMP

  • IP

  • RIP

  • TCP

  • UDP

The Indy global stack instance.
GStack: TIdStack  = nil;
Unit
IdStack
Description
GStack is a global variable that represents the current stack object for the Indy component set. GStack should never be changed, and is created and destroyed internally as needed. This is mentioned only because there a few useful low-level properties and methods.

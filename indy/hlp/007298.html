TIdDirItemTypeSet
Represents attributes that applied to an entry in a formatted directory listing.
Description
TIdDirItemType is set that can contain TIdDirItemType values representing the attributes applied to a TIdFTPListItem object instance in a formatted directory listing. TIdDirItemTypeSet is the type used to represent the TIdFTPListItem.ItemType property.

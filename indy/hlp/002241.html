IdHash128AsHex
See Also
Builds the hexadecial representation of a T4x4LongWordRecord value.
Return Value
String - 32 byte hexdecimal represention for the 128-bit hash value.
Description
IdHash128AsHex is a String function that returns the T4x4LongWordRecord value in AValue expressed as 32 hexadecimal digits. IdHash128AsHex iterates over the members in AValue constructing the hexadecimal representation in the return value using IdHexDigits.

EIdNotAllBytesSent
Hierarchy, See Also, Unit: IdSocketHandle
Exception for an incomplete transmission.
EIdNotAllBytesSent = class(EIdSocketHandleError)
Unit
IdSocketHandle
Description
EIdNotAllBytesSent is an EIdSocketHandleError exception descendant raised when TIdSocketHandle.SendTo is unable to send all bytes in a transmission to the peer connection.

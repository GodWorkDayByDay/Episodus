Unit: IdHTTPHeaderInfo.pas

IdHTTPHeaderInfo.pas contains classes and methods needed to implement support for HTTP message headers, as described in the Internet Standards document:

  • RFC 2616 - Hypertext Transfer Protocol version 1.1 [HTTP/1.1] (http://www.rfc-editor.org/rfc/rfc2616.txt)

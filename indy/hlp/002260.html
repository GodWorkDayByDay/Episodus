IdInBufCacheSizeDefault
See Also, Unit: IdTCPConnection
Default value for the TIdManagedBuffer.PackReadedSize property.
IdInBufCacheSizeDefault = 32 * 1024;
Unit
IdTCPConnection
Description
IdInBufCacheSizeDefault is an Integer constant that represents the default values assigned to the TIdManagedBuffer.PackReadedSize property. IdInBufCacheSizeDefault is also used when a a zero or negative value is addigned to TIdManagedBuffer.PackReadedSize.

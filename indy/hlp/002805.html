Unit: IdRFCReply.pas

IdRFCReply.pas contains classes and methods needed to implement an RFC Reply that consists of a numeric result code and associated message text in accordance with RFC specifications. RFC Replies are used by Servers and command handlers for server sessions to communicate responses in the form needed by specific protocols.

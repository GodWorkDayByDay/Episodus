IdBlockCipherBlockSizeDefault
See Also, Unit: IdBlockCipherIntercept
Represents the default block size for TIdBlockCipherIntercept send and receive operations.
IdBlockCipherBlockSizeDefault = 16;
Unit
IdBlockCipherIntercept
Description
IdBlockCipherBlockSizeDefault is a constant integer value that represents the default block size to use in TIdBlockCipherIntercept methods that send or receive data. IdBlockCipherBlockSizeDefault is the default value used for the TIdBlockCipherIntercept.BlockSize property as assigned in the TIdBlockCipherIntercept.Create method.

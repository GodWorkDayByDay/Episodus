TIdQOTDGetEvent
See Also, Unit: IdQotdServer
Specifies an event type for TIdQuotdServer commands.
TIdQOTDGetEvent = procedure ( Thread: TIdPeerThread ) of object;
Unit
IdQotdServer
Parameters
Thread
Connection generating the event.
Description
TIdQOTDGetEvent is an event type that allows TIdQOTDServer to respond to the OnCommandQOTD event. Thread is the thread for the client connection.

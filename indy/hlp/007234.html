Bytes
Provides access to the Byte values for the CRC checksum.
Return Value
Byte - Value in the index position.
Description
Bytes is a read-only Integer-indexed Byte property that provides access to the individual byte values for the calculated CRC checksum value. Index represents the Byte value and ranges from 0 to Size-1.

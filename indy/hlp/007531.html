TIdMappedPop3Thread
Hierarchy, Properties, Methods, Events, See Also, Unit: IdMappedPortTCP
Thread class for connections to a POP3 port-based proxy server.
TIdMappedPop3Thread = class(TIdMappedTelnetThread)
Unit
IdMappedPortTCP
Description
TIdMappedPop3Thread is a TIdMappedTelnetThread descendant thet implements the thread class created for client connections to a POP3 port-based proxy server. TIdMappedPop3Thread differs from the ancestor in the way that it internally established the outbound connection to the remote system for the proxied connection.

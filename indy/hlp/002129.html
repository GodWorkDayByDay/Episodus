IdBlockCipherBlockSizeMax
See Also, Unit: IdBlockCipherIntercept
Represents the maximum block size to allow for TIdBlockCipherIntercept send and receive operations.
IdBlockCipherBlockSizeMax = 256;
Unit
IdBlockCipherIntercept
Description
IdBlockCipherBlockSizeMax is a constant integer value that represents the maximum block size to allow for TIdBlockCipherIntercept send and receive operations. IdBlockCipherBlockSizeMax is used to determine the upper limit for the TIdBlockCipherIntercept.BlockSize property.

EIdTextInvalidCount
Hierarchy, See Also, Unit: IdMessage
Indicates an error creating text message parts.
EIdTextInvalidCount = class(EIdMessageException)
Unit
IdMessage
Description
EIdTextInvalidCount is a EIdMessageException exception descendant that identifies when a error has occurred while creating TIdText portions of TIdMessage during message transmission. EIdTextInvalidCount is the exception raised when heazders are generated for a message in TIdMessage.GenerateHeader.

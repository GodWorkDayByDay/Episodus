sLineBreak
See Also, Unit: IdGlobal
Indicates the default line end character sequence.
sLineBreak = EOL;
Unit
IdGlobal
Description
sLineBreak is a constant value that indicates the default line end character sequence to use when readingor writing Internet protocol transmission data. sLineBreak is defined for VCL versions prior to VCL 6. sLineBreak is already defined in Delphi 6 and Kylix.

Class type reference used to represent the Indy protocol stack.
TIdStackClass = class of TIdStack;
Unit
IdStack
Description
TIdStackClass is a TIdStack class type that represents the class family to use when creating the platform-specific representation of the protocol stack. TIdStackClass is the type used to represent the global variable GStackClass.

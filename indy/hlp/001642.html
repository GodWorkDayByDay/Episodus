ID_NETWORKCLASS
Unit: IdNetworkCalculator
Default network class for IP addresses in TIdNetworkCalculator instances.
ID_NETWORKCLASS = ID_NET_CLASS_A;
Unit
IdNetworkCalculator
Description
ID_NETWORKCLASS is a constant value that represents the default TNetworkClass value assigned to the TIdNetworkCalculator.NetworkClass property in the constructor for the object instance.

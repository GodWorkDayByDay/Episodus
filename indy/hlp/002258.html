IdIMAP4.pas
See Also, Classes, Types, Constants
Unit: IdIMAP4.pas

IdIMAP4.pas contains constants, types, classes, and methods needed to implement an IMAP4 (Internet Message Access Protocol version 4.1) client, as described in the Internet standards document:

  • RFC 2060 - Internet Message Access Protocol version 4.1 (http://www.rfc-editor.org/rfc/rfc2060.txt)

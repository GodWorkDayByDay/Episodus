EIdObjectTypeNotSupported
Hierarchy, Unit: IdTCPConnection
Exception raised when Capture can not support an object as a Destination.
EIdObjectTypeNotSupported = class(EIdTCPConnectionError)
Unit
IdTCPConnection
Description
This exception is raised when Capture is passed an object in the ADest parameter which Capture does not support. Capture only supports the following destination objects: TStream and TStrings.

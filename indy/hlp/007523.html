TIdMailBoxAttributesSet
See Also, Unit: IdMailBox
Represents a set of mailbox attributes for an IMAP4 mailbox.
TIdMailBoxAttributesSet = set of TIdMailBoxAttributes;
Unit
IdMailBox
Description
TIdMailBoxAttributesSet is a Set type that represents the TIdMailBoxAttributes applied to a given TIdMailBox instance. TIdMailBoxAttributesSet is used as an argument to TIdIMAP4.ParseMailBoxAttributeString to retrieve the attributes applied to the currently selected IMAP4 mailbox.

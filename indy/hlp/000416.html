EIdNNTPNoOnNewGroupsList
Hierarchy, See Also, Unit: IdNNTP
NNTP exception for retrieval of the new Newsgroups list.
EIdNNTPNoOnNewGroupsList = class(EIdNNTPException)
Unit
IdNNTP
Description
EIdNNTPNoOnNewGroupsList is an EIdNNTPException exception descendant raised when TIdNNTP.GetNewGroupsList is called and the TIdNNTP.OnNewGroupsList event handler has not been assigned.

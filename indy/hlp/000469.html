EIdSSLProtocolReplyError
Hierarchy, Properties, Methods, See Also, Unit: IdException
Represents an exception raised during Secure HTTP server access.
EIdSSLProtocolReplyError = class(EIdProtocolReplyError)
Unit
IdException
Description
EIdSSLProtocolReplyError is a EIdProtocolReplyError descendnant that represents the exception raised as a result of a protocol error when conecting to an HTTP server using the Secure HTTP protocol. EIdSSLProtocolReplyError can be rasised when an HTTP client attempts to connect a Secure HTTP server using a proxy connection, or when executing a HTTP command that points to a Secure HTTP server URL.

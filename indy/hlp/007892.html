TIdVCardEmbeddedObject
Hierarchy, Properties, Methods, See Also, Unit: IdVCard
Encapsulates an embedded object in a VCard.
TIdVCardEmbeddedObject = class(TPersistent)
Unit
IdVCard
Description
The TIdVCardEmbeddedObject object encapsulates an embedded object in a VCard such as a graphic, sound, or public-key (for encryption).
TIdVCardEmbeddedObject.Base64Encoded
TIdVCardEmbeddedObject
Indicates that EmbeddedData is represented Base64-encoded.
property Base64Encoded: Boolean;
Description
Base64Encoded is a Boolean indicator is True if EmbeddedData is represented as by Base64-encoded values.
TIdVCardEmbeddedObject.EmbeddedData
TIdVCardEmbeddedObject
ASCII or Binary data for the VCard.
property EmbeddedData: TStrings;
Description
EmbeddedData represents a storage slot for ASCII or binary data in the VCard. EmbeddedData is usually stored in an encoded format such as BASE64, although some keys may not require decoding. You can check data encoding by examining the Base64Encoded property.
TIdVCardEmbeddedObject.ObjectType
TIdVCardEmbeddedObject
Indicates the file or key type for the VCard.
property ObjectType: String;
Description
ObjectType indicates the type of media, such as the file type or key type.
TIdVCardEmbeddedObject.ObjectURL
TIdVCardEmbeddedObject
URL for object in a VCard.
property ObjectURL: String;
Description
ObjectURL is the Universal Resource Locator where the object is located if it is NOT embedded in the VCard.

Description
Create is the constructor for the object instance. Create relies on the inherited Create method, and initializes the EmbeddData TStringList property.

TIdVCardEmbeddedObject, See Also
Frees an object instance.

Destroy is the destructor for an object instance. Destroy clears and releases the EmbeddedData property prior to calling the inherited Destroy method.

IP_WATCH_HIST_FILENAME
See Also, Unit: IdIPWatch
Identifies the default history storage file.
IP_WATCH_HIST_FILENAME = 'iphist.dat';
Unit
IdIPWatch
Description
IP_WATCH_HIST_FILENAME is a constant to define the default filename used to save and restore the history list maintained by TIdIPWatch. The default value for IP_WATCH_HIST_FILENAME is IPHIST.DAT.

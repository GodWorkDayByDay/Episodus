TIdNNTPOnXOver
Unit: IdNNTPServer
TIdNNTPOnXOver = procedure (AThread: TIdNNTPThread; const AMsgFirst: Integer; const AMsgLast: Integer) of object;
Unit
IdNNTPServer
Parameters
AThread
NNTP client for the XOVER exchange.
VMsgFirst
First message number for the XOVER exchange.
VMsgLast
Last message number for the XOVER exchange.
Description
TIdNNTPOnXOver is an event type signalled when an NNTP client connection is used to perform an NNTTP XOVER protocol exchange.

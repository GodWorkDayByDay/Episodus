TIdIMAP4SearchKeyArray
See Also, Unit: IdIMAP4
Represents a group of search keys to use for an IMAP4 search request.
TIdIMAP4SearchKeyArray = array of TIdIMAP4SearchKey;
Unit
IdIMAP4
Description
TIdIMAP4SearchKeyArray is an array of TIdIMAP4SearchKey values that represents a group of search keys to use for an IMAP4 search request. TIdIMAP4SearchKeyArray can be used to represent multiple selections from the defined data items available to the IMAP4 SEARCH command.

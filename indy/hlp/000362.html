EIdCompressionException
Hierarchy, See Also, Unit: IdCompressionIntercept
Ancestor for ZLib library exceptions.
EIdCompressionException = class(EIdException)
Unit
IdCompressionIntercept
Description
EIdCompressionException is an EIdException descendant that acts as the abstract ancestor for exceptions that can be raised during use of the ZLib compression and decompression library. EIdCompressionException is not normally raised by an application.
